一些命令

发表于 2021-01-27


一些经常忘记但常用的命令

1.hexo篇

hexo new <br>新建一篇博客

hexo generate
生成静态文件

hexo deploy
部署网站

2.git篇

git log + git reset
以上两个命令可以协同将代码回滚到具体哪次提交

git revert
git revert也是回滚，但是他是回滚具体提交，区分与reset回退到版本。revert会在版本基础上再多一次提交记录。用于记录回滚的代码。

git clean -fd
回滚代码后，一些新建的文件会依然存在你的本地，状态显示是在被git add .之前。 一个一个手动删除太麻烦。这个命令可以帮助删除执行回滚后仅在你本地存在的文件。

git branch | grep -v “master” | xargs git branch -D
删除本地除指定分支外的所有分支，用于清空本地分支

3.linux篇

history
查看操作历史
