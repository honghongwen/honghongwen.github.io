Linux常用知识整理

1.查看配置

  • free -m

    首先拿到服务器后,至少得了解下服务器的配置如何,几核几g等等,free -m就是查看内存大小的一组命令，当然你也可以登录阿里云等或直接问同事.

    当然 free 直接显示的就是kb为单位的内存大小。-m既为换算成兆为单位，如上。这就是传统意义上的16g的一个服务器。

  • cat /etc/redhat-release

    如果系统预制安装的也为centos，对应的去/etc下查看版本信息。当然也有其他命令，不同系统，那么多命令，总有一款适合你。如

    lsb_release -a

    cat /proc/version

    等。

2.一些最简单linux命令

  • ll 查看文件夹下目录
  • ll -a 查看文件夹下包含隐藏目录
  • cd / 去往根目录
  • cd 去往当前用户所在目录
  • pwd 查看当前所在路径
  • mv {source} {target} 移动 也可用作重命名
  • cp {source} {target} 拷贝
  • rm -rf {source} !!!删除指定目录下文件 ！嗯 小白注意 需要了解到linux本身就为文件夹. 如果 {source} 是 / 根目录 那就完完了 所谓的删库跑路就是这个命令了。
  • mkdir {name} 创建文件夹
  • touch {name} 创建文件
  • cat 文件 查看文件全部信息
  • tail -n 1000f {xxx.logs} 查看文件的后一千行
  • su 切换管理员权限
